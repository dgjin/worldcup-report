# 2026 FIFA 世界杯实时战报 · 用户说明

> 网址：https://worldcup-report.pages.dev

---

## 功能概览

本站覆盖 2026 FIFA 世界杯（美国 / 加拿大 / 墨西哥，48 队赛制）全程赛事，提供八大功能板块，数据每分钟自动更新。

---

## 八大板块详解

### 1. 积分榜

小组赛 A–L 共 12 组实时积分表。

- **绿色**：各组前二，直接晋级淘汰赛
- **金色**：最佳第三（12 组各第三名中积分最高的 8 支晋级）
- **灰色**：已淘汰

页面顶部「今日速递」汇总当天全部比赛，显示已赛 / 进行中 / 待踢三种状态，并列出每场进球明细（含点球、乌龙标注）。

---

### 2. 射手榜

展示本届世界杯进球数最多的球员。

- 前三名以金 / 银 / 铜领奖台形式突出显示
- 每位球员附带真实头像、所属国旗、进球进度条
- 点球数单独标注，一目了然

---

### 3. 淘汰赛

分三个子区域：

| 子区域 | 内容 |
|---|---|
| 近期赛程 | 未来 6 天按日分组的比赛计划 |
| 对阵图 | 从 32 强到决赛的完整淘汰赛支架（桌面端 SVG / 移动端按轮折叠） |
| 晋级推演 | 根据当前积分自动推算哪些球队已确定晋级、哪些仍在争夺最佳第三席位 |

---

### 4. 冠军预测

基于 **9 维加权模型**计算每支球队的夺冠概率：

**实时数据维度（38%）**
- 小组赛表现、进攻火力、防守稳固、近期状态

**历史知识库维度（62%）**
- FIFA 最新排名、阵容市值（Transfermarkt 2026.06）、世界杯历史底蕴、伤病情况、大赛经验

页面同时提供**球迷投票**功能：

1. 从列表中选择你预测的冠军 / 亚军 / 季军（三个名次不可重复）
2. 点击「提交」完成投票
3. 每个浏览器只能投票一次，结果实时聚合展示

---

### 5. 球队资料卡

查看 48 支参赛球队的详细信息：

- 积分 / 胜平负 / 进球 / 净胜球四宫格数据
- 已赛对阵战绩（胜 W / 平 D / 负 L）
- 核心球员及本队射手
- 完整阵容名单（按门将 / 后卫 / 中场 / 前锋分组，含年龄）

在左侧小组折叠面板中点击队名即可切换查看。

---

### 6. 数据图表

用可视化图表呈现赛事统计：

- 本届世界杯关键指标（总进球、已赛场次、场均进球、最大分差）
- 各小组进球数柱状对比（≥12 球金色高亮）
- 各轮次进球走势
- 射手榜 Top 10 横向条形图

---

### 7. 战报

每场比赛自动生成中文赛事简报，支持按标签筛选：

| 标签 | 条件 |
|---|---|
| 爆冷 | 弱队击败传统强队 |
| 血洗 | 净胜球 ≥ 4 |
| 大胜 | 净胜球 = 3 |
| 一球小胜 | 净胜球 = 1 |
| 进球大战 | 总进球 ≥ 5 |
| 握手言和 | 有进球的平局 |
| 闷平 | 0:0 |

---

### 8. 精彩瞬间

汇集多家体育媒体（NewsAPI / ABC News / USA Today / AP News）的世界杯现场图集。

- **瀑布流**展示，向下滚动自动加载更多
- 点击任意图片进入**全屏灯箱**，可键盘左右键或触摸滑动翻页
- 支持为喜欢的照片**点赞**

---

## 界面操作

### 导航切换

- **桌面端**：页面顶部标签栏，点击切换板块
- **移动端**：页面底部导航栏，图标对应各板块

### 主题切换

页面右上角（桌面）或顶部栏提供**三态主题**切换：

- 浅色
- 深色
- 自动（跟随系统）

### 数据刷新

顶部状态栏显示当前数据来源（实时 / 缓存 / 离线快照），点击**刷新按钮**可立即拉取最新数据。

比赛进行中时，数据每 **15 秒**自动更新；比赛间隙每 **60 秒**更新一次。切换到其他应用再返回时自动触发刷新。

### 球迷交流区

位于积分榜页面内。在输入框中发表留言，内容将实时滚动显示在跑马灯中。

## 数据说明

| 数据类型 | 来源 |
|---|---|
| 积分 / 比分 / 赛程 | football-data.org 实时 API |
| 进球明细 | 人工维护（football-data 免费档不含进球字段） |
| 球员头像 | TheSportsDB |
| 国旗图标 | flagcdn.com |
| 阵容身价 | Transfermarkt（2026 年 6 月数据） |
| 精彩图集 | NewsAPI / ABC News / USA Today / AP News |

当实时 API 不可用时，系统依次回退到 Supabase 缓存（约 2 分钟延迟）或本地离线快照，页面始终可用，顶部会提示当前数据源。

## 常见问题

**积分或比分没有更新？**
点击页面顶部的刷新按钮，或检查网络连接后重新打开页面。

**球迷投票提交后没有反应？**
每个浏览器只能投票一次，若已投过将不显示投票入口。

**图集加载很慢？**
图片来自外部媒体网站，加载速度受对方服务器影响，向下滚动会自动加载后续内容。

**淘汰赛对阵图显示"待定"？**
小组赛尚未全部结束，晋级名额仍在推算中，对阵将在成绩确定后自动填入。
